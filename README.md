### What is it?

[Fishing for dummies](https://chrisdouce.github.io/Fishing-for-dummies/) is a website for easily calculating Sea Creature probability weights on Hypixel Skyblock.

Select your location, fishing stats, and other conditions to alter what your current SC pool. Modifiers will alter weights and display those changes accordingly. You can also filter to hide unwanted SCs, pooling the weights to what you desire.

If there are any issues like out of date information or website issues, [create an issue here](https://github.com/Chrisdouce/Fishing-for-dummies/issues)
